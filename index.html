СБЛИЖЕНИЕ

Главная Новости О нас

Приложения для людей с ограниченными возможностями здоровья по слуху

Пользователь использует язык жестов, который фиксирует камера устройства, камера сопоставляет жест с имеющейся базой данных и переводит его в текст, также пользователь может включить озвучивание текста. У пользователя есть возможность перевода голосовых сообщений в текст с последующим сохранением и перевод текста в жесты (с помощью данной функции можно проводить обучение для глухих людей, мероприятия, праздники). В ходе развития проекта планируется введение искусственного интеллекта. Это позволит пользователям формулировать из отдельных слов логичные предложения (многие люди с ОВЗ, имеющие врожденную глухоту, не могут точно и верно формулировать предложения, что является еще одним барьером).
